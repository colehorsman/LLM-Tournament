# Agent Evaluation Rubric

**Evaluator**: Dr. Zero (BLIND Evaluation)  
**Protocol**: No knowledge of which LLM generated the application

---

## Scoring Dimensions (20 points each)

### Technical Depth (0-20)

| Score | Criteria |
|-------|----------|
| 18-20 | Specific CVEs, attack chains, production-ready code examples |
| 14-17 | Real expertise with concrete examples |
| 10-13 | Surface-level knowledge, correct but generic |
| 5-9 | Buzzwords without substance |
| 0-4 | Incorrect or missing |

### Practical Relevance (0-20)

| Score | Criteria |
|-------|----------|
| 18-20 | Step-by-step guidance a CISO could implement today |
| 14-17 | Specific recommendations with clear next steps |
| 10-13 | Generic advice that's technically correct |
| 5-9 | Vague platitudes without actionable guidance |
| 0-4 | Impractical or harmful recommendations |

### AI/Agentic Focus (0-20)

| Score | Criteria |
|-------|----------|
| 18-20 | Deep expertise on agent identity, A2A auth, agent-specific risks |
| 14-17 | Good awareness of agentic security landscape |
| 10-13 | Mentioned but not deep |
| 5-9 | Superficial or outdated understanding |
| 0-4 | Missing entirely |

### Actionability (0-20)

| Score | Criteria |
|-------|----------|
| 18-20 | Production-ready with clear implementation path |
| 14-17 | Clear next steps with reasonable effort |
| 10-13 | Needs significant work to implement |
| 5-9 | Unclear how to proceed |
| 0-4 | Not actionable |

### Team Integration (0-20)

| Score | Criteria |
|-------|----------|
| 18-20 | Real workflows with data flow, not just skill lists |
| 14-17 | Good collaboration patterns defined |
| 10-13 | Skills listed but workflows unclear |
| 5-9 | Isolated, no team awareness |
| 0-4 | Conflicts with team structure |

---

## Verdict Thresholds

| Score | Verdict | Action |
|-------|---------|--------|
| 85-100 | **HIRE** | Onboard immediately |
| 78-84 | **PROBATION** | One more round with feedback |
| 60-77 | **REVISE** | Major improvements needed |
| <60 | **REJECT** | Not suitable for role |

---

## AI Slop Detection (Automatic Penalties)

Automatic flags for:

- Generic openings: "In today's digital landscape..."
- Buzzword soup: "AI-powered", "next-gen", "holistic"
- Vague advice: "prioritize security", "stay vigilant"
- Corporate speak: "leverage", "synergy", "stakeholders"
- Fear mongering without data
- Format crimes: >3 hashtags, >2 emojis
